# Emasiya_Final


## '이마시야'란?
'이마시야'는 오픈 SW 프로젝트 수업에서 진행한, 이화 여자 대학교 주변 맛집을 소개하는 웹 프로젝트입니다. 코로나19 시기에 학교에 오지 못했던 벗들을 위해 리뷰와 평점을 남겨 맛집 데이터베이스를 만들 수 있습니다.<br/>

## 개발 도구
- goormide
- HTML , CSS, JS
- Flask
- Firebase<br/>


## 개발 기간
2022.09 - 2022.12<br/>
